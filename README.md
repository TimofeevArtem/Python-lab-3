# Лабораторная работа 3

* **Цель** Написать Сортировки, функции фибоначчи и факториала и структуру данных
** Бибилотеки** *pytest*

* **Чему я научилсяя**
  - Написание различных сортировок
  - Использование класса
  - Делать стек

### Алгоритм решения
  1. Пользователь вводит с какой структурой он хочет работать(Стек, сортировки или др)
  2. Пользователь следует указаниям программы и вводит то, с чем хочет работать
  3. Программа выводит ответ(в случае с факториалом - факториал введеного пользователем числа, в случае сортировки - отсортированный список)

## Запуск
Из корневой дирректории вписать в терминал
```python src/main.py```
